Summary

After studying this chapter you should achieve the following outcomes:

  1. Understanding of the purpose and potential applications of Linked Data.
  2. Some familiarity with background technologies and standards such as HTTP, RDF, Turtle, OWL, and SPARQL.
  3. Ability to formulate correct RDF statements in Turtle, using either full URIs or abbreviations.
  4. Ability to use some basic resources of RDFS, such as "rdfs:label" and "rdfs:subClassOf".
  5. Understanding of the basic principles for exposing Linked Data.
  6. Some familiarity with web sites already available which provide services based on Linked Data.
